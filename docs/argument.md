#### /api/v1/arguments/argument [POST] - Створити аргумент

***Вхідні параметри***

```json
{
  "name": ""
}
```

#### /api/v1/arguments/argument [GET] - Получити список аргументів

***Вихідні параметри***

```json
[
  {
    "id": 1,
    "name": ""
  }
]
```

#### /api/v1/arguments/argument/{argument_id} [GET] - Получити аргумент по id

***Вихідні параметри***

```json
{
  "id": 1,
  "name": ""
}
```
